# 🌍 如何切换语言

## 快速指南

### 1️⃣ 启动项目

```bash
cd apps\web
bun run dev
```

### 2️⃣ 打开浏览器

访问: **http://localhost:3000**

### 3️⃣ 切换语言

在页面右上角找到语言切换器:

```
[OpenCut Logo]  Blog  Contributors  Projects  [🌐 语言] [🌙 主题]
                                                   ↑
                                              点击这里!
```

点击地球图标 🌐,会出现下拉菜单:
- ✓ **中文** (当前选择会显示勾选标记)
- **English**

### 4️⃣ 查看效果

选择中文后,页面立即变成中文:

#### 英文界面 → 中文界面

| 英文 | 中文 |
|-----|------|
| Blog | 博客 |
| Contributors | 贡献者 |
| Projects | 项目 |
| The Open Source Video Editor | 开源视频编辑器 |
| Try early beta | 试用早期版本 |
| Resources | 资源 |
| Company | 公司 |
| Roadmap | 路线图 |
| Privacy policy | 隐私政策 |
| Terms of use | 使用条款 |

## 🔧 开发者说明

### 文件位置
- 翻译文件: `apps/web/src/lib/i18n/locales/`
- 语言切换器: `apps/web/src/components/locale-switcher.tsx`
- 状态管理: `apps/web/src/stores/locale-store.ts`

### 在代码中使用

```typescript
import { useLocaleStore } from "@/stores/locale-store";

function MyComponent() {
  const { t, locale, setLocale } = useLocaleStore();
  
  return (
    <>
      <h1>{t.hero.title}</h1>
      <p>当前语言: {locale}</p>
      <button onClick={() => setLocale({ locale: "zh" })}>
        切换到中文
      </button>
    </>
  );
}
```
