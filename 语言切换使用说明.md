# OpenCut 多语言功能使用说明

## ✨ 新功能

OpenCut 现在支持多语言！目前支持：
- 🇺🇸 English (英语)
- 🇨🇳 中文简体

## 🔄 如何切换语言

### 方法一：通过顶部导航栏
1. 在任何页面的右上角，找到 **🌐 地球图标** 按钮
2. 点击该按钮打开语言选择菜单
3. 选择您想要的语言（English 或 中文）
4. 页面会立即切换到选择的语言

### 方法二：自动记忆
- 您选择的语言会自动保存在浏览器中
- 下次访问时会自动使用您上次选择的语言
- 即使关闭浏览器后再次打开，语言设置仍然保留

## 📍 语言切换按钮位置

### 首页
- 位置：右上角导航栏，主题切换按钮旁边
- 显示：地球图标 🌐

### 项目页面
- 位置：右上角导航栏，主题切换按钮旁边
- 显示：地球图标 🌐

## 🔧 技术实现

### 翻译文件位置
```
apps/web/src/lib/i18n/
├── index.ts                # 国际化配置
├── locales/
│   ├── zh.ts              # 中文翻译
│   └── en.ts              # 英文翻译
└── README.md              # 开发者文档
```

### 在组件中使用翻译
```tsx
"use client";

import { useLocaleStore } from "@/stores/locale-store";

export function MyComponent() {
  const { t } = useLocaleStore();
  
  return (
    <div>
      <h1>{t.hero.title}</h1>
      <p>{t.hero.subtitle}</p>
    </div>
  );
}
```

### 语言状态管理
- 使用 Zustand 进行全局状态管理
- 使用 localStorage 持久化存储用户选择
- 存储键：`opencut-locale`

## 📝 为翻译做贡献

如果您想为 OpenCut 添加新语言或改进现有翻译，请：
1. Fork 项目仓库
2. 创建新的语言文件（如 `ja.ts`）
3. 参考 `en.ts` 或 `zh.ts` 的结构添加翻译
4. 在 `apps/web/src/lib/i18n/index.ts` 中注册新语言
5. 在 `apps/web/src/components/locale-switcher.tsx` 中添加语言选项
6. 提交 Pull Request

## ❓ 常见问题

**Q: 语言切换后没有立即生效？**
A: 确保页面已经完全加载。如果仍有问题，尝试刷新页面（Ctrl + F5）。

**Q: 某些文本没有翻译？**
A: 我们正在逐步完善翻译。如发现未翻译的文本，欢迎提交 issue 或 PR。

**Q: 如何添加新语言？**
A: 请参考上面的"为翻译做贡献"章节，或查看 `apps/web/src/lib/i18n/README.md` 文件。

**Q: 语言设置会在不同设备间同步吗？**
A: 目前语言设置仅存储在本地浏览器中。未来可能会添加账户同步功能。
